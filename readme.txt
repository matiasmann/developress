DeveloPress is a lightweight WordPress theme Framework with native support for WooCommerce and Gutenberg.

== Description ==

We follow WordPress coding standards wich allows compatiblity with many plugins by default.
DeveloPress is fully responsive, uses valid HTML/CSS. 
There are included +5 widget areas, +3 navigation locations, +3 sidebar layouts, dropdown menus (click or hover) and navigation color presets. 
Learn more and check out our powerful premium version at https://developress.org. Contributions at our GitHub repo: https://github.com/developressag

Learn more and check out our [powerful premium version](https://developress.org).

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "DeveloPress"
3. Install and Activate

== Frequently Asked Questions ==

= Is DeveloPress Free? =
Yes! DeveloPress is a free and open source theme, and always will be.

= Do you offer support? =
Definitely. We offer support for the free theme in the [WordPress.org forums](https://wordpress.org/support/theme/developress).

Premium customers have access to our very own [support forum](https://developress.org/support).

We try to answer all questions - free or premium - within 48 hours. We know whats is like to be stucked with something =)

= What languages do you speak? =
We speak english and spanish natively. We are based in Buenos Aires, Argentina.

= Where can I find documentation? =
DeveloPress has extensive documentation you can find [here](https://docs.developress.com).

= Does DeveloPress have a premium version? =
Yes. DeveloPress has a premium plugin which extends the available options in the theme. More colors, more layouts, more options for WooCommerce and Awesome Support from our team.

You can learn more about DP Premium [here](https://developress.org/premium).

= Where can I find the theme options? =
All of our options can be found in the Customizer in 'Appearance > Customize'.

= Does DeveloPress have any widget areas? =
DeveloPress has up to +5 widget areas which you can add widgets to in Appearance > Widgets.

= How can I make my site look like your screenshot? =
If you want to replicate the screenshot you see on WordPress.org, please refer to [this article](https://docs.developress.com/article/replicating-the-screenshot/).
